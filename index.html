Alle Befehle aus den Abschnitten 1 bis 10

Diese Tabelle listet alle gängigen Linux-Befehle aus den verschiedenen Abschnitten des Handbuchs sowie grundlegende Systembefehle auf. Sie dient als schnelle Referenz für die Arbeit mit Linux-Systemen.

Befehl Beschreibung
ls Listet Dateien und Verzeichnisse im aktuellen Verzeichnis auf.
cd Wechselt in ein anderes Verzeichnis.
pwd Zeigt den aktuellen Pfad des Arbeitsverzeichnisses an.
cp Kopiert Dateien und Verzeichnisse.
mv Verschiebt oder benennt Dateien und Verzeichnisse um.
rm Löscht Dateien und Verzeichnisse.
touch Erstellt eine neue leere Datei oder ändert das Datum einer bestehenden Datei.
chmod Ändert die Berechtigungen einer Datei oder eines Verzeichnisses.
chown Ändert den Besitzer einer Datei oder eines Verzeichnisses.
chgrp Ändert die Gruppe einer Datei oder eines Verzeichnisses.
find Findet Dateien und Verzeichnisse basierend auf verschiedenen Kriterien.
grep Sucht nach Textmustern in Dateien.
useradd Erstellt einen neuen Benutzer.
usermod Ändert ein Benutzerkonto.
userdel Löscht ein Benutzerkonto.
groupadd Erstellt eine neue Gruppe.
groupdel Löscht eine Gruppe.
chage Ändert das Ablaufdatum des Passworts eines Benutzers.
passwd Ändert das Passwort eines Benutzers.
apt update Aktualisiert die Paketdatenbank (Debian, Ubuntu).
apt upgrade Aktualisiert alle installierten Pakete auf die neueste Version.
apt install Installiert ein neues Paket (Debian, Ubuntu).
apt remove Entfernt ein Paket (Debian, Ubuntu).
dnf install Installiert ein Paket (Fedora).
yum update Aktualisiert alle Pakete (CentOS, RHEL).
yum install Installiert ein Paket (CentOS, RHEL).
ifconfig Zeigt oder konfiguriert Netzwerkschnittstellen (deprecated in vielen Distributionen, ersetzt durch ip).
ip a Zeigt alle Netzwerkinterfaces und deren IP-Adressen an.
ping Testet die Netzwerkverbindung zu einem anderen Host.
traceroute Verfolgt den Pfad von Paketen zu einem Zielhost.
netstat Zeigt Netzwerkverbindungen, Routing-Tabellen und Netzwerkstatistiken an.
ss Zeigt Informationen zu Sockets und Netzwerkverbindungen an (moderner Ersatz für netstat).
top Zeigt eine Echtzeitübersicht über die CPU- und Speichernutzung sowie laufende Prozesse.
htop Erweiterte Version von top mit einer benutzerfreundlicheren Oberfläche.
free Zeigt Informationen über den verfügbaren und verwendeten Arbeitsspeicher und Swap-Speicher.
vmstat Zeigt Informationen über den virtuellen Speicher und Systemaktivitäten an.
ps Zeigt die derzeit laufenden Prozesse an.
kill Beendet einen laufenden Prozess anhand seiner PID.
systemctl Verwaltet systemweite Dienste unter systemd (Starten, Stoppen, Neustarten von Diensten).
journalctl Zeigt Systemprotokolle an, die von systemd verwaltet werden.
fdisk Werkzeug zur Partitionierung von Festplatten (MBR-Partitionen).
parted Partitionierungstool, das sowohl MBR als auch GPT unterstützt.
mount Binde ein Dateisystem oder eine Partition in das Verzeichnisbaum ein.
umount Trenne eine Partition oder ein Dateisystem vom Dateisystem.
rsync Synchronisiert Dateien und Verzeichnisse zwischen verschiedenen Orten.
tar Erstellt und extrahiert Archivdateien (z. B. .tar.gz).
dd Erstellt Festplattenabbilder und kopiert Daten auf Blockebene.
cryptsetup Verwaltet Festplattenverschlüsselung mit LUKS.
iptables Verwalte Netzwerk-Firewall-Regeln auf der Basis von iptables.
ufw Benutzerfreundliches Frontend für die Konfiguration von iptables.
firewalld Verwalte Firewall-Regeln unter firewalld.
ssh Verbindet sich sicher mit einem entfernten Linux-Server über das SSH-Protokoll.
auditd Überwacht und protokolliert Sicherheitsereignisse im System.
systemctl reboot Startet das System neu.
systemctl poweroff Fährt das System herunter.
grub-mkconfig Erstellt eine neue GRUB-Konfigurationsdatei.
